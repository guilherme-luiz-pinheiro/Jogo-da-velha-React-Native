# Jogo da Velha - React Native com Expo

## 📌 Descrição
Este é um jogo da velha desenvolvido em **React Native** utilizando **Expo**. O jogo permite que dois jogadores alternem entre os símbolos "X" e "O", e verifica automaticamente se houve uma vitória ou empate.

## 🚀 Tecnologias Utilizadas
- React Native
- Expo
- Hooks do React (`useState`)
- Componentes do React Native (`View`, `Text`, `Image`, `TouchableOpacity`, `Alert`)

## 📥 Instalação e Execução
### 1️⃣ Clone o repositório
```
git clone https://github.com/seu-usuario/JogoDaVelha.git
cd JogoDaVelha
```

### 2️⃣ Instale as dependências
Caso ainda não tenha o Expo instalado:
```
npm install -g expo-cli
```
Agora instale as dependências do projeto:
```
npm install
```

### 3️⃣ Execute o projeto
```
expo start
```
Abra o app no simulador ou no seu celular escaneando o QR Code com o aplicativo **Expo Go**.

## 🎮 Como Jogar
1. O primeiro jogador é representado pelo **X** e o segundo pelo **O**.
2. Os jogadores se alternam clicando nos espaços vazios do tabuleiro.
3. O jogo verifica automaticamente se houve uma **vitória** ou um **empate**.
4. Para reiniciar, clique no botão **Limpar**.

## 🖼️ Assets
As imagens do jogo (X e O) devem estar na pasta `./assets/`.
